# Change Log

All notable changes to the "yapi" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [0.0.1] - 2025-01-02

### Added

- Multi-protocol support for HTTP, gRPC, GraphQL, and TCP requests
- Real-time validation with inline diagnostics and error highlighting
- Live response panel with syntax-highlighted JSON responses
- Quick example insertion command
- Keyboard shortcut `Cmd+Enter` / `Ctrl+Enter` to run requests
- Execution timing display for request completion times
- YAML syntax error detection with line-level diagnostics
- Protocol-specific validation (e.g., gRPC requires `service` and `rpc`)
